Day 05 - Linux Basics

What is Linux?

Linux is a free and open-source operating system used on servers, cloud platforms, smartphones, and by cybersecurity professionals. Many penetration testing tools, including Kali Linux, are built on Linux.

---

Why is Linux Important in Cybersecurity?

- Most web servers run Linux.
- Kali Linux is the most popular operating system for ethical hacking.
- Linux provides powerful command-line tools.
- It offers better control over system security.
- Many cybersecurity tools are designed for Linux.

---

Common Linux Distributions

Ubuntu

A beginner-friendly Linux distribution widely used for learning and development.

Kali Linux

A Linux distribution designed for penetration testing and ethical hacking.

Debian

A stable and reliable Linux distribution used by many servers.

Fedora

A modern Linux distribution with the latest features.

---

Basic Linux Commands

pwd

Displays the current working directory.

Example:

pwd

ls

Lists files and folders in the current directory.

Example:

ls

cd

Changes the current directory.

Example:

cd Documents

mkdir

Creates a new directory.

Example:

mkdir Cybersecurity

touch

Creates a new file.

Example:

touch notes.txt

cp

Copies files or folders.

Example:

cp notes.txt backup.txt

mv

Moves or renames files.

Example:

mv notes.txt Day5-notes.txt

rm

Deletes files.

Example:

rm notes.txt

clear

Clears the terminal screen.

Example:

clear

---

Best Practices

- Double-check commands before pressing Enter.
- Avoid using the "rm" command carelessly.
- Keep your Linux system updated.
- Use strong passwords.
- Learn command-line navigation daily.

---

Key Takeaways

- Linux is the preferred operating system for many cybersecurity professionals.
- Learning terminal commands is an essential cybersecurity skill.
- Practice is the best way to become comfortable with Linux.
