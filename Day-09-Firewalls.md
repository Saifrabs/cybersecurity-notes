# Day 09 - Firewalls

## What is a Firewall?

A Firewall is a network security device or software that monitors, filters, and controls incoming and outgoing network traffic based on predefined security rules.

Its main purpose is to block unauthorized access while allowing legitimate communication.

---

## Why are Firewalls Important?

Firewalls help to:

- Protect computers and networks from hackers.
- Block unauthorized access.
- Prevent malware from entering a network.
- Monitor network traffic.
- Improve overall network security.

---

## How Does a Firewall Work?

A firewall examines every piece of data (called a packet) entering or leaving a network.

It compares each packet against predefined security rules.

- If the traffic is safe, it is allowed.
- If the traffic is suspicious or unauthorized, it is blocked.

---

## Types of Firewalls

### 1. Packet Filtering Firewall

Checks each data packet based on IP address, port number, and protocol.

**Advantage:**
- Fast and simple.

**Disadvantage:**
- Cannot inspect the contents of packets.

---

### 2. Stateful Inspection Firewall

Monitors active connections and makes decisions based on the state of network traffic.

**Advantage:**
- More secure than packet filtering.

---

### 3. Proxy Firewall

Acts as an intermediary between users and the internet.

**Advantage:**
- Hides internal network details.
- Provides additional security.

---

### 4. Next-Generation Firewall (NGFW)

A modern firewall that includes advanced security features.

Features include:

- Deep Packet Inspection (DPI)
- Intrusion Prevention System (IPS)
- Application Control
- Malware Detection
- Web Filtering

---

## Hardware vs Software Firewalls

### Hardware Firewall

A physical device placed between a network and the internet.

**Example:**
A firewall built into a home or office router.

---

### Software Firewall

A program installed on a computer to protect that specific device.

**Examples:**

- Windows Defender Firewall
- Linux UFW (Uncomplicated Firewall)

---

## Advantages of Firewalls

- Prevent unauthorized access.
- Block malicious traffic.
- Improve network security.
- Monitor network activity.
- Reduce cyberattack risks.

---

## Limitations of Firewalls

- Cannot stop attacks caused by careless users.
- Cannot prevent phishing if users click malicious links.
- Requires regular updates and proper configuration.

---

## Best Practices

- Keep your firewall enabled at all times.
- Update firewall software regularly.
- Only allow trusted applications.
- Monitor firewall logs.
- Combine firewalls with antivirus software and Multi-Factor Authentication (MFA).

---

## Real-Life Example

A hacker attempts to connect to a company's internal server from the internet.

The firewall detects that the connection is not authorized and blocks the request before the attacker gains access.

---

## Summary

- Firewalls protect networks by controlling traffic.
- They allow legitimate traffic and block suspicious traffic.
- Different firewall types provide different levels of protection.
- Firewalls are an essential part of every cybersecurity strategy.

---

## Key Takeaways

- Firewalls act as the first line of defense.
- Every organization should use a properly configured firewall.
- Firewalls reduce cyber risks but should be combined with other security measures.
